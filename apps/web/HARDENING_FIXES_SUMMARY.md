999);\n\n// WEEK: Last 7 days\nstartDate.setDate(istTime.getDate() - 6);  // 7 days inclusive\n\n// MONTH: Last 30 days\nstartDate.setDate(istTime.getDate() - 29);  // 30 days inclusive\n```\n\n###
